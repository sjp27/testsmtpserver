# NAME
**testsmtpserver** - Test SMTP server

# SYNOPSIS
**testsmtpserver**
<*port*>
<*security (NONE, SSL, TLS)*>
[*fail (command to stop responding e.g. AUTH)*]

# DESCRIPTION
**testsmtpserver**
Test SMTP server that displays email data and can simulate a failure at a given command - does not respond.
